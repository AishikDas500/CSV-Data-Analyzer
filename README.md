# CSV Data Analyzer with Linear Regression

## Description
A Python-based tool that allows users to analyze any CSV file, visualize relationships between variables, and perform linear regression to predict outcomes.

## Tech Stack
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Load any CSV file
- Select independent and dependent variables
- Visualize data using scatter plots
- Perform linear regression
- Predict values based on input
- Export predictions to a new CSV file

## How to Run
1. Install dependencies:
   pip install pandas matplotlib scikit-learn

2. Run the program:
   python csv_analyzer.py

3. Enter file path and follow menu options
